## คอนเซปต์ที่เข้าใจแล้ว (Concepts Understood)

> เพิ่มคอนเซปต์ที่ผู้เรียนอธิบายกลับได้เองแล้ว

- **dev server + `npm run dev`** — รันเว็บบนเครื่องตัวเองดูที่ `localhost:3000`, หยุดด้วย `Ctrl + C`
- **Warning ≠ Error** — เหลือง = เตือนแต่ทำงานต่อได้ / แดง/failed = พังต้องแก้
- **Git 3 ด่าน** — `git add` (ใส่กล่อง) → `git commit -m` (ปิดผนึก+ติดป้าย, เซฟในเครื่อง) → push (ขึ้น GitHub)
- **`command not found` / `not recognized`** — มักเป็นเรื่อง PATH ยังไม่อัปเดต → เปิด terminal ใหม่
- **Gmail `+alias`** — `name+xxx@gmail.com` เข้ากล่องเดิม แต่เว็บมองเป็นคนละอีเมล
